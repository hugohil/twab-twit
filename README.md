# twab twit

## Installation

```shell
$ npm i
$ cp config.js.exampe config.js
```

Then put your credentials into `config.js` file

## Usage

```shell
$ node index.js
```

Then make a `POST` request to your server like this: `http://localhost:4242/update/twitterHandle=@hugohil&twabURL=hugohil.ninja/twab`.
